1. Создать среду.Делается один раз на весь проект. Если в процессе работы с проектом среда создалась заново, значит, вы попытались "активировать" её не из той директории
  2. Активировать среду. Та же команда (в той папке, где есть Pipfile)
    • Windows: py -m pipenv shell
    • MacOS: python3 -m pipenv shell
    Среда либо аквируется сама, если VSCode её "нашел", либо её приходится активировать всякий раз, когда открываете новый терминал
  3. Создать проект (django-admin startproject НАЗВАНИЕ) - один раз за работу над всем проектом
  4. Создать приложение (python manage.py startapp mainpage) - приложений может быть несколько и они могут называться по разному!
  5. Создать файл с путями к приложению и записать его в глобальный urls
  6. Зарегистрировать приложение в settings.py INSTALLED_APPS
  7. создать директорию с шаблонами и шаблонами в приложении templates а в ней mainpage
  8. создать функцию, отображающую шаблонам во views.py 
    def main(request):
    render(
        request,               # так будет всегда
        'mainpage/main.html',  # путь к шаблону
        # здесь будут данные!
    )
  9. Прописать путь к функции path('', views.main, name='mainpage'),
  10. Обратиться по пути
  11. Запустить сервер - каждый раз, когда он упал. python manage.py runserver
